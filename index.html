各功能使用方式與說明
🚀 快速開始
1
設定個人資料
前往「設定」填寫姓名、學號、科系、入學年度、修業年限，並設定當前學期。所有資料存在瀏覽器，清除後會消失。
2
新增課程
點擊「＋ 新增課程」，填入學期、課名、學分、分類、成績與狀態。也可至「匯入課程」用 CSV 批次匯入歷年修課紀錄。
3
設定畢業門檻
前往「畢業門檻」，設定總學分需求、時數需求、英文門檻類型與分數，系統會自動計算是否達標。
4
使用成績預估器
在「成績管理 → 成績預估器」，為每門課新增預估器，設定各項配分比例，系統可計算預估分數並反推期末目標分。
5
匯出報告
點擊「匯出報告」可產生包含學分總覽、各類進度、畢業門檻的 PDF 報告。
📋
學分管理系統
  • 新增 / 編輯 / 刪除課程
  • 分類：校定必修、系必修、專業選修、通識、外系選修、外語選修
  • 課程狀態：已通過、修課中、未通過、未修
  • 自動計算各類學分統計
  • 學期篩選、關鍵字搜尋
  • 選課模擬功能：試算修完後學分
📌 選課模擬模式下新增的課程不會計入正式統計
🎓
畢業門檻管理
  • 設定總學分、時數、英文門檻
  • 支援 TOEIC / GEPT / 校內測驗 / 修課替代
  • 自動判定各項是否達標
  • 可新增自訂門檻項目（開關型 / 進度型）
  • 個人資料同步（英文類型、時數）
📌 總學分數由學分管理頁面自動計算，無需手動輸入
📊
成績管理與預估
  • 自動計算學期 GPA 與累積 GPA
  • 成績趨勢圖（各學期平均）
  • 成績預估器：自訂各項評分比例
  • 反推功能：目標分 → 期末需幾分
  • 系統自動套用 4.0 / 百分制換算
🗓
行事曆與課表
  • 月曆視圖，最多顯示 2 筆事件
  • 點擊日期快速新增事件
  • 支援匯入 .ics 行事曆檔案
  • 今日課表與本週課表
  • 課程時間可設定星期與節次
📌 手機版每格最多顯示 2 筆，點擊日期可看完整內容
📌 重要提醒
💾 所有資料儲存在瀏覽器 localStorage，清除瀏覽器資料後將消失，建議定期匯出 PDF 備份。
⚠ 選課模擬的課程不會計入正式學分統計，請在確定選課後再加入正式課程。
📄 PDF 匯出建議在桌面版瀏覽器操作以獲得最佳效果。中文字體若顯示異常，請確認已連線網路。
🔮 成績預估器各項比例總和須為 100%，系統會提示不符的情況。
